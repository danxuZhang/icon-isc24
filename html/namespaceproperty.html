Calculates the fall speed of hydrometeors.

This function calculates the fall speed of hydrometeors based on their density and a set of parameters. The fall speed is determined using a power law relationship.

Parameters
[in]densityThe density of the hydrometeors (kg/m^3).
[in]paramsAn array of parameters used in the fall speed calculation.
  • params[0]: Coefficient of the power law relationship.
  • params[1]: Exponent of the power law relationship.
  • params[2]: Offset added to the density.
Returns
The fall speed of the hydrometeors (m/s).

Calculates the snow lambda parameter.

This function calculates the snow lambda parameter based on the ambient density, snow specific mass, and snow number concentration. The lambda parameter is used in the size distribution of snow particles. If the snow specific mass is less than or equal to graupel_ct::qmin, a constant value of lmd_0 is returned.

Parameters
[in]rhoAmbient density (kg/m^3).
[in]qsSnow specific mass (kg/kg).
[in]nsSnow number concentration (1/kg).
Returns
The snow lambda parameter (1/m).
